# Funding Tiers & Financial Roadmap

Technical decisions are aligned with these funding milestones.

## Tier 1: Pre-Seed ($1.5M)
- **Focus**: Minimum Viable Ecosystem (MVE).
- **Milestones**:
  - Conxius Wallet (Beta)
  - Conxian Gateway (Fusion)
  - Stacks-native Swap interface
- **Runway**: 12-18 months.

## Tier 2: Seed ($5M - $8M)
- **Focus**: Scaling and User Acquisition.
- **Milestones**:
  - Full Institutional API rollout
  - Mobile Wallet v1.0 (Mainnet)
  - $50M TVL across protocol
- **Infrastructure**: Transition to multi-region cloud deployment.

## Tier 3: Series A ($20M+)
- **Focus**: Global Expansion.
- **Milestones**:
  - Cross-border settlement rails
  - $500M TVL
- **Infrastructure**: Begin hybrid cloud/on-prem research for critical nodes.

## Tier 4: Series B/C ($100M+)
- **Focus**: Dominance in the Bitcoin-DeFi market.
- **Milestones**: $1B+ TVL.
